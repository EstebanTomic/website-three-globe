## General info
This project is my personal and first 3D website implemented with Three JS.

## Technologies
Project is created with:
* npm version: 9.6.0
* vite: 4.1.0
* three: 0.150.1
* gsap: 3.11.4

## Setup
To run this project, install it locally using npm:
```
$ npm install
$ npm run dev
```
